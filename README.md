## How to run locally.
1. Install NodeJS latest LTS version.
2. Install yarn.
```sh
npm install -g yarn
```
3. Clone this repository and `cd` into it.
4. Install dependencies.
```sh
yarn install
```
5. Start local server.
```sh
yarn start
```
